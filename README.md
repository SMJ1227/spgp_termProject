# spgp_termProject
 
# High Concept
	* 플레이어는 몰려오는 적을 정확한 타이밍에 공격하여 제거하면서 최대한 오랜 시간 살아남는다.

# 핵심 메카닉
	* 플레이어는 위 아래로 현재 위치한 위치를 바꿀 수 있고, 적을 공격할 수 있다.
	* 적은 위치에 맞춰 플레이어 쪽으로 다가온다.

# 개발 범위
	* 플레이어 이동 범위: 3개 이상의 층
	* 몬스터: 3개 이상의 적
 * 맵: 단일 맵
 * 몬스터와 플레이어간의 충돌처리
 * 일시정지, 메인화면과 같은 기본 UI
